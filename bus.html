Bus Discipline Referral Automation Guide

A beginner-friendly setup guide for replacing triplicate paper bus write-ups with a simple Google Form that automatically creates a PDF referral and emails the right staff.

What this system does

This setup is designed so the transportation director does not have to run a mail merge manually. The workflow is: fill out the form, submit it, and the system does the rest.

1. Submit Form A driver or transportation staff member completes a Google Form.
2. Generate PDF Apps Script copies a Google Doc template and fills in the referral details.
3. Notify Staff The finished PDF is saved in Drive and emailed to selected staff.
Student privacy reminder: bus discipline referrals may include student information. Use a restricted Drive folder, limit access, and follow district policy before emailing or sharing records.

This guide assumes you are using Google Workspace, Google Forms, Google Sheets, Google Docs, Drive, and Apps Script.

Apps Script code

Open the Form response spreadsheet. Go to Extensions → Apps Script. Delete any starter code and paste this entire script.

Full Apps Script code 
/*******************************************************
 * Bus Discipline Referral PDF Generator
 *
 * What it does:
 * - Runs when a Google Form response is submitted
 * - Creates a completed Google Doc from a template
 * - Exports the completed referral as a PDF
 * - Saves the PDF in a Drive folder
 * - Writes the PDF link back to the response Sheet
 * - Optionally emails the PDF to staff
 *******************************************************/

const CONFIG = {
  // Usually this is "Form Responses 1"
  RESPONSE_SHEET_NAME: 'Form Responses 1',

  // Paste your Google Docs template ID here
  DOC_TEMPLATE_ID: 'PASTE_TEMPLATE_DOC_ID_HERE',

  // Paste your Drive folder ID here
  PDF_FOLDER_ID: 'PASTE_PDF_FOLDER_ID_HERE',

  // Optional. Leave blank unless you want temporary Google Docs in a separate folder.
  TEMP_DOC_FOLDER_ID: '',

  // Set to true to email the generated PDF
  SEND_EMAIL: true,

  // Default recipients. Separate multiple emails with commas.
  DEFAULT_EMAIL_TO: 'transportation@example.org',

  // Optional CC recipients
  DEFAULT_EMAIL_CC: '',

  // Optional BCC recipients
  DEFAULT_EMAIL_BCC: '',

  // If true, the temporary Google Doc copy is moved to trash after PDF is created.
  // The final PDF is kept.
  DELETE_TEMP_DOC_AFTER_PDF: true,

  // Date/time zone. Usually this can stay as-is.
  TIMEZONE: Session.getScriptTimeZone(),

  // Optional school-based routing.
  // The School value must exactly match the School dropdown/answer in the Form.
  SCHOOL_EMAIL_ROUTING: {
    // Example:
    // 'Middle School': 'middleschoolprincipal@example.org',
    // 'High School': 'highschoolprincipal@example.org'
  }
};


/**
 * Adds a custom menu when the spreadsheet opens.
 */
function onOpen() {
  SpreadsheetApp.getUi()
    .createMenu('Bus Referrals')
    .addItem('1. Run Setup', 'setupSystem')
    .addSeparator()
    .addItem('Process Selected Row', 'processSelectedRow')
    .addItem('Process All Unprocessed Rows', 'processAllUnprocessedRows')
    .addToUi();
}


/**
 * Run this one time after pasting the script.
 * It adds tracking columns and creates the form submit trigger.
 */
function setupSystem() {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const sheet = getResponseSheet_();

  ensureTrackingColumns_(sheet);

  const triggers = ScriptApp.getProjectTriggers();
  const alreadyExists = triggers.some(trigger =>
    trigger.getHandlerFunction() === 'onFormSubmitBusReferral'
  );

  if (!alreadyExists) {
    ScriptApp.newTrigger('onFormSubmitBusReferral')
      .forSpreadsheet(ss)
      .onFormSubmit()
      .create();
  }

  SpreadsheetApp.getUi().alert(
    'Setup complete.\n\nThe script will now process new bus referral form submissions automatically.'
  );
}


/**
 * This runs automatically when the Google Form is submitted.
 */
function onFormSubmitBusReferral(e) {
  const lock = LockService.getScriptLock();

  try {
    lock.waitLock(30000);

    if (!e || !e.range) {
      throw new Error('No form submit event data was found. Use Process Selected Row for manual testing.');
    }

    const sheet = e.range.getSheet();

    if (sheet.getName() !== CONFIG.RESPONSE_SHEET_NAME) {
      return;
    }

    const row = e.range.getRow();
    processReferralRow_(sheet, row, false);

  } catch (err) {
    console.error(err);
  } finally {
    lock.releaseLock();
  }
}


/**
 * Manual test function.
 * Select any response row, then run:
 * Bus Referrals → Process Selected Row
 */
function processSelectedRow() {
  const sheet = getResponseSheet_();
  const row = sheet.getActiveRange().getRow();

  if (row <= 1) {
    SpreadsheetApp.getUi().alert('Please select a response row, not the header row.');
    return;
  }

  processReferralRow_(sheet, row, true);

  SpreadsheetApp.getUi().alert('Selected row processed.');
}


/**
 * Processes every row that does not already have a PDF link.
 */
function processAllUnprocessedRows() {
  const sheet = getResponseSheet_();
  ensureTrackingColumns_(sheet);

  const lastRow = sheet.getLastRow();

  if (lastRow <= 1) {
    SpreadsheetApp.getUi().alert('No response rows found.');
    return;
  }

  let processedCount = 0;

  for (let row = 2; row <= lastRow; row++) {
    const data = getRowObject_(sheet, row);

    if (!data['PDF Link']) {
      processReferralRow_(sheet, row, false);
      processedCount++;
    }
  }

  SpreadsheetApp.getUi().alert(`Done. Processed ${processedCount} row(s).`);
}


/**
 * Main worker function.
 */
function processReferralRow_(sheet, row, forceReprocess) {
  ensureTrackingColumns_(sheet);

  const headerMap = getHeaderMap_(sheet);
  const data = getRowObject_(sheet, row);

  try {
    if (data['PDF Link'] && !forceReprocess) {
      return;
    }

    const referralId = data['Referral ID'] || createReferralId_(row, data);
    data['Referral ID'] = referralId;

    writeCell_(sheet, row, headerMap, 'Referral ID', referralId);
    writeCell_(sheet, row, headerMap, 'Error', '');

    const result = createReferralPdf_(data);

    writeCell_(sheet, row, headerMap, 'PDF Link', result.pdfUrl);
    writeCell_(sheet, row, headerMap, 'PDF File ID', result.pdfFileId);
    writeCell_(sheet, row, headerMap, 'Generated At', new Date());

    if (CONFIG.SEND_EMAIL) {
      sendReferralEmail_(data, result.pdfBlob, result.pdfUrl);
      writeCell_(sheet, row, headerMap, 'Email Status', `Sent ${new Date()}`);
    } else {
      writeCell_(sheet, row, headerMap, 'Email Status', 'Email disabled');
    }

  } catch (err) {
    writeCell_(sheet, row, headerMap, 'Error', err.message);
    console.error(err);
    throw err;
  }
}


/**
 * Creates the Google Doc from template, fills it, exports as PDF,
 * and saves the PDF to Drive.
 */
function createReferralPdf_(data) {
  validateConfig_();

  const templateFile = DriveApp.getFileById(CONFIG.DOC_TEMPLATE_ID);
  const pdfFolder = DriveApp.getFolderById(CONFIG.PDF_FOLDER_ID);
  const tempFolder = CONFIG.TEMP_DOC_FOLDER_ID
    ? DriveApp.getFolderById(CONFIG.TEMP_DOC_FOLDER_ID)
    : pdfFolder;

  const safeStudentName = makeSafeFilename_(data['Student Name'] || 'Unknown Student');
  const safeDate = makeSafeFilename_(data['Date of Incident'] || formatDate_(new Date()));
  const referralId = data['Referral ID'];

  const baseFileName = `Bus Referral - ${safeStudentName} - ${safeDate} - ${referralId}`;

  const docCopy = templateFile.makeCopy(baseFileName, tempFolder);
  const doc = DocumentApp.openById(docCopy.getId());
  const body = doc.getBody();

  // Replace every {{Header Name}} placeholder using the Sheet/Form headers.
  Object.keys(data).forEach(key => {
    const tag = `{{${key}}}`;
    replaceTag_(body, tag, valueToDisplayString_(data[key]));
  });

  // Helpful extra tags.
  replaceTag_(body, '{{Today}}', formatDate_(new Date()));
  replaceTag_(body, '{{Generated At}}', formatDateTime_(new Date()));

  doc.saveAndClose();

  const pdfBlob = DriveApp.getFileById(docCopy.getId())
    .getAs(MimeType.PDF)
    .setName(`${baseFileName}.pdf`);

  const pdfFile = pdfFolder.createFile(pdfBlob);

  if (CONFIG.DELETE_TEMP_DOC_AFTER_PDF) {
    docCopy.setTrashed(true);
  }

  return {
    pdfFileId: pdfFile.getId(),
    pdfUrl: pdfFile.getUrl(),
    pdfBlob: pdfBlob
  };
}


/**
 * Sends the referral email.
 */
function sendReferralEmail_(data, pdfBlob, pdfUrl) {
  const to = getEmailRecipients_(data);

  if (!to) {
    throw new Error('No email recipient was found. Check DEFAULT_EMAIL_TO or SCHOOL_EMAIL_ROUTING.');
  }

  const studentName = data['Student Name'] || 'Unknown Student';
  const school = data['School'] || 'Unknown School';
  const incidentDate = valueToDisplayString_(data['Date of Incident']) || 'Unknown Date';
  const referralId = data['Referral ID'] || '';

  const subject = `Bus Discipline Referral: ${studentName} - ${incidentDate}`;

  const body =
`A new bus discipline referral has been submitted.

Referral ID: ${referralId}
Student: ${studentName}
School: ${school}
Date of Incident: ${incidentDate}
Bus Number: ${data['Bus Number'] || ''}
Route: ${data['Route'] || ''}
Driver: ${data['Driver Name'] || ''}

PDF:
${pdfUrl}

This message was generated automatically from the bus discipline referral system.`;

  MailApp.sendEmail({
    to: to,
    cc: CONFIG.DEFAULT_EMAIL_CC,
    bcc: CONFIG.DEFAULT_EMAIL_BCC,
    subject: subject,
    body: body,
    attachments: [pdfBlob]
  });
}


/**
 * Gets default recipients plus optional school-specific recipients.
 */
function getEmailRecipients_(data) {
  const recipients = [];

  if (CONFIG.DEFAULT_EMAIL_TO) {
    recipients.push(CONFIG.DEFAULT_EMAIL_TO);
  }

  const school = data['School'];

  if (
    school &&
    CONFIG.SCHOOL_EMAIL_ROUTING &&
    CONFIG.SCHOOL_EMAIL_ROUTING[school]
  ) {
    recipients.push(CONFIG.SCHOOL_EMAIL_ROUTING[school]);
  }

  return recipients
    .filter(Boolean)
    .join(',');
}


/**
 * Adds tracking columns if they do not already exist.
 */
function ensureTrackingColumns_(sheet) {
  const neededColumns = [
    'Referral ID',
    'PDF Link',
    'PDF File ID',
    'Generated At',
    'Email Status',
    'Error'
  ];

  const headers = getHeaders_(sheet);

  neededColumns.forEach(columnName => {
    if (!headers.includes(columnName)) {
      sheet.getRange(1, sheet.getLastColumn() + 1).setValue(columnName);
    }
  });
}


/**
 * Reads the current row into an object using the header row.
 */
function getRowObject_(sheet, row) {
  const headers = getHeaders_(sheet);
  const values = sheet.getRange(row, 1, 1, headers.length).getValues()[0];

  const data = {};

  headers.forEach((header, index) => {
    if (header) {
      data[header] = values[index];
    }
  });

  return data;
}


/**
 * Gets the response sheet.
 */
function getResponseSheet_() {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const sheet = ss.getSheetByName(CONFIG.RESPONSE_SHEET_NAME);

  if (!sheet) {
    throw new Error(`Could not find sheet named "${CONFIG.RESPONSE_SHEET_NAME}". Check CONFIG.RESPONSE_SHEET_NAME.`);
  }

  return sheet;
}


/**
 * Gets headers from row 1.
 */
function getHeaders_(sheet) {
  const lastColumn = sheet.getLastColumn();

  if (lastColumn < 1) {
    return [];
  }

  return sheet.getRange(1, 1, 1, lastColumn).getValues()[0];
}


/**
 * Creates a map of header names to column numbers.
 */
function getHeaderMap_(sheet) {
  const headers = getHeaders_(sheet);
  const map = {};

  headers.forEach((header, index) => {
    if (header) {
      map[header] = index + 1;
    }
  });

  return map;
}


/**
 * Writes a value to a specific column by header name.
 */
function writeCell_(sheet, row, headerMap, headerName, value) {
  if (!headerMap[headerName]) {
    ensureTrackingColumns_(sheet);
    headerMap = getHeaderMap_(sheet);
  }

  sheet.getRange(row, headerMap[headerName]).setValue(value);
}


/**
 * Creates a readable unique referral ID.
 */
function createReferralId_(row, data) {
  const dateValue = data['Date of Incident'] || new Date();
  const datePart = formatDateForId_(dateValue);
  const rowPart = String(row).padStart(4, '0');

  return `BUS-${datePart}-${rowPart}`;
}


/**
 * Replaces a template tag safely.
 */
function replaceTag_(body, tag, value) {
  const escapedTag = escapeRegex_(tag);
  const safeValue = sanitizeReplacement_(value);
  body.replaceText(escapedTag, safeValue);
}


/**
 * Escapes regex characters for replaceText.
 */
function escapeRegex_(text) {
  return String(text).replace(/[.*+?^${}()|[\]\\]/g, '\\$&');
}


/**
 * Makes replacement text safer for Apps Script replaceText.
 */
function sanitizeReplacement_(value) {
  return String(value || '')
    .replace(/\\/g, '\\\\')
    .replace(/\$/g, '\\$');
}


/**
 * Converts dates and other values into clean display text.
 */
function valueToDisplayString_(value) {
  if (value instanceof Date) {
    return formatDateTime_(value);
  }

  if (value === null || value === undefined) {
    return '';
  }

  return String(value);
}


/**
 * Formats a date.
 */
function formatDate_(value) {
  const date = value instanceof Date ? value : new Date(value);

  if (isNaN(date.getTime())) {
    return String(value || '');
  }

  return Utilities.formatDate(date, CONFIG.TIMEZONE, 'MM/dd/yyyy');
}


/**
 * Formats a date/time.
 */
function formatDateTime_(value) {
  const date = value instanceof Date ? value : new Date(value);

  if (isNaN(date.getTime())) {
    return String(value || '');
  }

  return Utilities.formatDate(date, CONFIG.TIMEZONE, 'MM/dd/yyyy h:mm a');
}


/**
 * Formats a date for referral IDs.
 */
function formatDateForId_(value) {
  const date = value instanceof Date ? value : new Date(value);

  if (isNaN(date.getTime())) {
    return Utilities.formatDate(new Date(), CONFIG.TIMEZONE, 'yyyyMMdd');
  }

  return Utilities.formatDate(date, CONFIG.TIMEZONE, 'yyyyMMdd');
}


/**
 * Makes text safe for filenames.
 */
function makeSafeFilename_(value) {
  return String(value || '')
    .replace(/[\\/:*?"<>|#%{}~&]/g, '-')
    .replace(/\s+/g, ' ')
    .trim()
    .substring(0, 80);
}


/**
 * Checks that required IDs are filled in.
 */
function validateConfig_() {
  if (!CONFIG.DOC_TEMPLATE_ID || CONFIG.DOC_TEMPLATE_ID === 'PASTE_TEMPLATE_DOC_ID_HERE') {
    throw new Error('Missing DOC_TEMPLATE_ID in CONFIG.');
  }

  if (!CONFIG.PDF_FOLDER_ID || CONFIG.PDF_FOLDER_ID === 'PASTE_PDF_FOLDER_ID_HERE') {
    throw new Error('Missing PDF_FOLDER_ID in CONFIG.');
  }
}

Test the system

Do not test with a real student first. Use a fake student and make sure the whole loop works.

  • Submit a fake Google Form response.
  • Open the response Sheet.
  • Look for the generated Referral ID.
  • Check the PDF Link column.
  • Open the Drive folder and confirm the PDF exists.
  • Check whether the email arrived.
  • Open the PDF and make sure all fields filled correctly.
Good sign: if the response Sheet has a PDF link, a generated timestamp, and no error message, the automation is working.

Troubleshooting

“Could not find sheet named Form Responses 1”

Your response sheet tab may have a different name. Look at the bottom tab in the spreadsheet and update this line:

Sheet name setting
RESPONSE_SHEET_NAME: 'Form Responses 1',

The PDF is created, but some fields are blank

The Google Doc template tag probably does not match the Form question exactly.

Example: if the Form question is Incident Description, the template tag must be {{Incident Description}}.

The email does not send

Check these two settings:

Email settings
SEND_EMAIL: true,
DEFAULT_EMAIL_TO: 'transportation@example.org',

The script asks for permissions

That is expected. It needs permission to read the Sheet, create Docs, save PDFs, and send email.

School routing does not work

The school name in the routing section must exactly match the school name submitted in the Form.

There is an error column message

Read the message in the Error column. Most issues are missing IDs, a renamed Sheet tab, or mismatched template tags.

Recommended rollout

Keep version one simple. The goal is trust, not every feature imaginable.

Start with this
  • One Form
  • One PDF template
  • One secure folder
  • Internal staff emails only
  • One week of fake testing
Add later
  • School-specific routing
  • Status dropdowns
  • Parent contact log
  • Repeat student dashboard
  • Monthly route trend report
Do not overbuild the first version. If the transportation director has to think about mail merge, folders, or scripts, the system is too complicated. Their workflow should be: open form, fill it out, submit.
