⚙ Setup & Connection

Connect to your Google Sheets backend to save and sync data across sessions.

📋 Setup Instructions
  1. Create a new Google Sheet.
  2. Go to Extensions → Apps Script and paste the contents of Code.gs.
  3. Click Deploy → New Deployment → Web App.
  4. Set Execute as: Me and Who has access: Anyone.
  5. Copy the Web App URL and paste it above, then click Connect.
  6. Go to the 👥 Classes tab and add your classes.
  7. You're ready — start planning on the 📋 Lesson Plans tab!
